## Arduino
ブートローダー書き込み済みのICを使ったスケッチ書き込み器

## 部品
* Arduino Bootlooder書き込み済み(I-12774)
* AE-TTL-232　USBシリアル変換キット(K-09951)
* 水晶発振子16MHz(P-08671)
* 積層セラミックコンデンサ－22pF,0.1uF
* 抵抗 10k,330
* 分割ロングピンソケット(C-05779)

## ブートローダー書き込みを行う場合
