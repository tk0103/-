# 概要（工事中）
* GPS受信機キットを用いて、経度・緯度・海抜を取得
* LCD表示
* ファームウェアアップデートを行う(ver0012の場合)

## 部品
* GPS受信機キット「みちびき」３機受信対応(K-09991)
* スイッチング・ダイオード1N4148(I-00941)
* LCDモジュール(P-10185)
* 半固定ボリューム
* 抵抗
