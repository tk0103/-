## Arduino
* 部屋にあるリモコンを統一。TV、扇風機、エアコン、照明
* トグルスイッチONで、扇風機ON・照明全灯

## 部品
* 赤外線リモコン受信モジュール(I-00614)
* 赤外線LED OS15LA5113A(I-12612)
* トグルスイッチ
* 抵抗

## 通信フォーマット
http://elm-chan.org/docs/ir_format.html
