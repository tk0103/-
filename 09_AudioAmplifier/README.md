# 概要

* HCU04のアンバッファ特性を利用

* 74HCU04AP(I-11328)
* ピンソケット14P(P-00006)
* スイッチ付き2連ボリュームA10k(P-05080)
* 2連ボリュームピッチ変換基盤(P-12311)
* 抵抗10,10k,33k,100k
* 積層セラミックコンデンサ0.1uF,0.33uF
* 電解コンデンサ 25V470mF
* 3.5mmステレオジャック基盤取付用(C-02460)
* 3.5mmステレオミニプラグオーディオケーブル(C-13083)
* 単4ケース
